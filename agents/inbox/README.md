# Agent Inbox README

Use inbox files for asynchronous agent-to-agent communication.

## Message Template

```md
## YYYY-MM-DD HH:MM — from <agent-id>

**Request:** What you need.

**Context:** Links to related tasks, handoffs, files, branches, or commits.

**Priority:** low | normal | high

**Needed by:** Optional.
```

Move resolved messages under a `Resolved Messages` section or into an archive file if this grows too large.
