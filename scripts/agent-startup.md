# Agent Startup Prompt

You are working in this repository as an autonomous project agent.

Before making changes:

1. Read `AGENTS.md`.
2. Read `context/PROJECT.md`.
3. Read `context/CURRENT.md`.
4. Read `context/DECISIONS.md`.
5. Read `context/CONVENTIONS.md`.
6. For design/frontend work, read `context/DESIGN.md` and `design/components.md`.
7. Check `agents/status.md`.
8. Check your inbox under `agents/inbox/`.
9. Review `tasks/active/`, `tasks/blocked/`, and `tasks/review/`.
10. Identify the highest-priority unblocked task.
11. If you claim a task, update its owner/status/branch and `agents/status.md`.
12. Work on only one task at a time.

Rules:

- Do not rely on hidden chat history as shared memory.
- The repo context is the source of truth.
- Do not edit files owned by another active task unless coordinated.
- Before stopping, update the task file and write a handoff in `handoffs/`.

After reading the context, summarize what you found and propose the next action before editing files.
