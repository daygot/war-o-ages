# Design — War O' Ages

## North Star
An illuminated war-atlas: aged parchment, walnut ink, sealing-wax red, manuscript gold, engraved Cinzel capitals, italic EB Garamond narration, heraldic shield crests, and a candle-lit war-room table.

## Voice
The game speaks as a chronicler. Use second person for the player and manuscript-like narration for events. Prefer copy like “Assemble a legion across history. Spin the wheel. Conquer the ages.” Avoid generic UI language like “Submit” or “OK.”

## Visual Guardrails
- Colors: parchment + walnut + sealing-wax red + manuscript gold, with region inks for meaning.
- Typography: Cinzel for structure, EB Garamond italic for flavor, Cinzel Decorative sparingly for wordmark.
- Surfaces: textured parchment, double rules, foxing, warm shadows; no glassmorphism.
- Motion: short ceremonial easing; no distracting spinners or bouncy modern motion.
- For current parity work, prefer preserving the approved desktop prototype structure/copy over speculative redesign.
