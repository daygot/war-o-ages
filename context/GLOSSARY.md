# Glossary — War O' Ages

- **War O' Ages** — Daily historical legion-building game and current project name.
- **Battle Across Time** — Former project name; archived only.
- **Wheel of Ages** — Two-ring draft wheel combining region and era.
- **The Commander / Strategist / General / Troops / Allies** — Five ranks in the player's legion.
- **Ideology** — Cause the legion fights for; affects scoring and narrative identity.
- **Daily Battlefield** — The themed battleground for the day.
- **Campaign Table** — Desktop command-center layout and canonical prototype surface.
- **War Council** — Phase where ideology and synergies are reviewed before battle.
- **Verdict** — Final result screen and share-card moment.
- **Illuminated war-atlas** — The aesthetic north star for the full product.
- **Chronicler voice** — Mock-epic in-world narration style.
